# Analise de Requisitos Clínica Saúde+

## Stakeholders
+ Paciente:

+ Recepcionista: 

+ Médico: 

## Requesitos Funcionais

+ O sitema deve permitir  cadrastro de planos de saúde definidos pela clínica.

+ O Sistema deve enviar notificações de confirmações de reservas.

+ O Sistema deve  permitir logins por meio de redes sociais.

+ Enviar uma solicitação de confirmação de presença na consulta.

+ O sistema desmarca consulta não confirmadas em até 24h antes da data marcada.

+ O pacicente pode acessar seu histórico de consultas e agendamentos por meio de login e senha.

+ O sistema deve permitir ao paciente fazer reservas.

+ O sistema deve notificar o paciente sobre as instruções sobre a consulta.

+ O sistema deve notificar o administrador em caso de falha de pagamento. 

+ O sistema deve aceitar pagamentos digitais (pix ou cartões).

+ O sistema deve notificar o cliente em caso de desmarcação por parte da clínica.

+ O sistema deve permitir  que o médico acesse a própria agenda. 

+ A recepcionista deve Aprovar solicitações de reservas. 

+ O sistema deve gerar relatórios de faturamentos. 

## Requesitos não Funcionais

+ O Sistema deve estar dísponivel para agendar, cancelar e consultar horários 24/7.

+ O sistema deve ser compatível com plataformas móveis.

+ O sitema deve mostrar a agenda em tempo real em diferentes dispositivos.

+ O sistema deve realizar backups automático diariamente.

+ O sistema deve armazenar os dados e o histórico de consultas do paciente.

+ O sistema deve carregar a página de cadastros em até 2 segundos.

+ O sistema deve ser compatível com os principais navegadores modernos.

+ O sistema deve suportar 1000 usuários simultâneamente.


## Regras de Negócios

+ Clientes que não possuem plano de saúde tem 20% de descontos.

+ O sistema deve aceitar somente planos de sáude pré-definidos pela clínica

+ Clientes que não confirme em até 24h antes da consulta terão sua consulta desmarcada.

+ Clientes menores de idade têm de estar acompanhados dos responsáveis.

+ Clientes que confirmam a consulta e não comparecem estão sujeitos a uma taxa de 10% de cobrança na próxima consulta marcada.
